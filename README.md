# Shoe Insole Pressure Sensor PCB
Schematic and Board Layout for Mizzou ECE Capstone Project

ESP 32 Code

## Repository Rules
When pushing KiCad files, do not push .lck files

## Toolchain
KiCad Version: 9.0

Arduino IDE

## Repository Structure
| Path                      | Description                     |
|---------------------------|---------------------------------|
| /Capstone-backups         | .zip backups                    |
| /Hierarchical Sheets      | .kicad_sch files                |
| /Spice                    | .kicad_sch files with Spice     |
| /Libs                     | Any imported IC models          |
| Capstone.kicad_sch        | Root schematic file             |
| Capstone.kicad_pro        | Project file                    |
| Capstone.kicad_pcb        | PCB file                        |
